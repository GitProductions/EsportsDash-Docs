# Default Overlay System

The Default Overlay comes pre-configured with every game configuration in Esports Dash. When you install a game config, this overlay automatically installs to OBS Studio as a scene collection, providing everything needed to start broadcasting immediately.

## Key Features
- Works with all supported game configurations
- Pre-configured scenes and sources
- Professional broadcast layout
- Easy customization options 


## Getting Started

<Steps>
### Install Game Configuration

</Steps>
